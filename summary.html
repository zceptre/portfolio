§ One — Profile

The developer Flow Pilot calls when an automation needs building.

I’m a developer based in the Philippines, building AI-augmented automation systems for international clients. As the on-call developer at Flow Pilot, I ship everything from custom CMS platforms to multi-stage n8n pipelines to AI agents that run business workflows end-to-end. My work spans WhatsApp commerce in the UAE, e-commerce CMS for a London furniture brand, B2B accounting automation, and full-stack admin dashboards on Railway.

I treat the gap between “a business has a problem” and “a system runs without supervision” as the only interesting territory. The tools I use — Playwright, OpenAI, Claude, n8n, Shopify API, React, PHP — are the means; the end is always the same: time that humans no longer have to spend on the work software should already be doing.

If it can be automated, I'll automate it. If it can't, I'll build the system that makes it automatable.
